# Deployment Roadmap

## Research-grade milestone
- deterministic benchmark
- reproducible prior training
- simulator results with realistic routes
- clear ablations
- open-source code and docs

## Translational milestone
- hardware integration on a known wheelchair base
- safety shell proven in repeated bench tests
- calibration reduced to less than 3 minutes
- clinician-facing profile report

## Product milestone
- industrial design
- battery and base integration
- diagnostic UI
- field updates
- cybersecurity review
- service process
- regulatory path by market

## Journal strategy
A strong paper would center on one of these claims:
1. capability-aware priors improve rapid personalization in continuous navigation
2. the safety shell plus calibration pipeline makes deployment practical
3. local feature abstraction preserves the benefits of the original tabular idea while scaling beyond a toy grid
